ASM Optimiser

ASM Optimizer

In the "Project ASM Optimiser" we tackle the challenge of optimizing quadratic cost functions using assembly language to implement gradient descent. Conducted as part of the Computer Organisation module at NUS, this project required us to develop a highly efficient assembly subroutine that minimizes the cost function by accurately calculating the optimal parameter settings. By mirroring the logic of the C function "optimize_c_rounded" and aiming to achieve even greater precision, our assembly implementation not only demonstrates the power of low-level programming in handling complex mathematical operations but also underscores the potential for significant performance enhancements in computational tasks. This work highlights our methodological approach, our adaptations to the traditional gradient descent algorithm, and our strategic use of assembly language to enhance computational efficiency and accuracy.

Alex to the Rescue

Alex to the Rescue

This project involves a tele-operated robot named "Alex," equipped with advanced sensor technologies to navigate a 7x5 grid maze. Key components include a forward-facing LIDAR for real-time mapping and an ultrasonic sensor for precise distance measurements. Using Hector SLAM, the robot manually plots the environment while identifying colored objects. Communication between a Raspberry Pi and Arduino is established through USART, allowing for efficient control and sensor data processing. Alex's teleoperation utilizes VNC and SSH protocols for remote control, and its movement and detection systems are tightly integrated to achieve real-time task execution within 8 minutes.
